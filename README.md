# Программа по шифрованию
## Лабораторная работа по криптографии. Столбцовый метод и шифр Виженера
Вариант 4. Шифрование методами перестановки и подстановки

## О проекте
Программа для шифрования и дешифрования текста на русском языке с помощью двух алгоритмов:

Столбцовый метод — перестановочный шифр  
Шифр Виженера с прогрессивным ключом — подстановочный шифр
- Все символы, кроме русских букв, автоматически игнорируются
- Поддержка буквы Ё
- Загрузка и сохранение .txt файлов

# 📘 Инструкция по использованию программы
## Шаг 1. Выберите алгоритм шифрования
Программа поддерживает два метода. Выберите нужный, заполнив соответствующие поля:

Столбцовый метод: Перестановка букв, ключ - слово, состоящее из букв английского алфавита  
Алгоритм Виженера: Подстановка, ключ - слово, состоящее из букв русского алфавита

## Шаг 2. Загрузите ключ и исходный текст  
Можно ввести вручную — напечатайте текст в поле Исходный текст.   
Можно загрузить из файла — нажмите кнопку [Загрузить данные из файла] и выберите .txt файл.

## Шаг 3. Шифрование и дешифрование
Используйте кнопки под полем ввода:

Кнопка [Зашифровать]:	преобразует исходный текст в зашифрованный  
Кнопка [Дешифровать]:	восстанавливает исходный текст из зашифрованного  
✅ После нажатия в поле Результат появится обработанный текст.


## Шаг 4. Сохраните результат
Нажмите [Сохранить результат в файл]: результат сохранится в выбранный вами файл.

## 📁 Работа с файлами (общие действия)  
[Загрузить все данные из файла] — загружает данные сразу для обоих алгоритмов.

[Сохранить все результаты в общий файл] — сохраняет результаты работы обеих методов в один текстовый файл.
